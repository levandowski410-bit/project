Проект: Анализ данных о сне и образе жизни
Описание проекта
Этот проект представляет собой комплексный анализ датасета "Sleep_health_and_lifestyle_dataset", содержащего информацию о здоровье сна и образе жизни 374 человек. Анализ выполнен с использованием Python (pandas, numpy, matplotlib, seaborn) и Yandex DataLens.
